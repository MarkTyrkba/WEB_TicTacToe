## Overview

I recently shared a CLI version of Tic Tac Toe, and now, I've put together this snazzy web-based version in just two days. It's a fun and interactive way to play the classic game right in your browser.

## How to Play

1. **Start Playing:** Click on the cells to make your move. X and O will take turns automatically.

2. **Game Status:** Keep an eye on the game status. When it's over, you'll know who the winner is.

3. **Reset the Board:** Want to play again? Hit the reset button and start fresh.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript

- **Backend:** Rust (Actix-Web)

- **Communication:** Fetch API for seamless communication between frontend and backend.

## Running Locally

1. Clone the repository:

    ```bash
    git clone https://github.com/MarkTyrkba/WEB_TicTacToe.git
    ```

2. Navigate to the project folder:

    ```bash
    cd WEB_TicTacToe
    ```

3. Start the server:

    ```bash
    cargo run
    ```

4. Open your browser and visit [http://localhost:8080](http://localhost:8080).

5. Play, enjoy, and have fun!
